# Support

Use GitHub Issues for reproducible bugs, feature requests, and documentation
problems.

Before filing a bug, include:

- LexiRay version or commit.
- macOS version.
- Xcode version if built locally.
- The provider and model involved, if relevant.
- Whether Accessibility, Screen Recording, and Automation permissions are
  granted to the workspace-built app.
- Clear reproduction steps and screenshots for UI issues.

Do not include API keys, private selected text, screenshots with sensitive data,
or full provider responses containing secrets.
